# React Native Layout - Login & Signup

A React Native project created with Expo featuring Login and Signup screens.

## Project Structure

```
react-native-layout/
├── screens/
│   ├── Login.js
│   └── Signup.js
├── assets/
│   └── images/
│       └── login.png (add your image here)
├── App.js
└── package.json
```

## Features

- Login screen with email and password inputs
- Signup screen with name, email, password, and confirm password inputs
- Ionicons integration for icons in inputs and buttons
- Clean and responsive layout
- Proper folder structure

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add login image:
   - Place your `login.png` image in `assets/images/` folder

3. Run the app:
```bash
npm start
```

## Screens

- **Login.js**: Login screen with image, email/password inputs, and login button
- **Signup.js**: Signup screen with name, email, password fields, and signup button

## Icons Used

- `mail-outline` - Email input
- `lock-closed-outline` - Password input
- `person-outline` - Name input
- `arrow-forward` - Login button
- `checkmark-circle` - Signup button
